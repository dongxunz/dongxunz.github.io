Marlin For OpenPNP固件参数配置

本文最后更新于：2025年4月14日 晚上

前言

Smoothieboard固件项目的README概述里说该项目适用于LPC17xx微控制器，但将固件烧录到BTT SKR V1.4（LPC1768）主板后，连最基础的移动都无法执行，串口会返回答复：printe pause，使用OpenPNP软件检测固件时也会打印已弃用的警告信息：WARNING:deprecated_MCU。
如果烧录到BTT SKR V1.4 Turbo（LPC1769）主板则是可以正常使用，暂不清楚导致该情况的具体原因。
而且该项目已经多年未维护，很多功能也不支持，所以本文章针对BTT SKR V1.4（LPC1768）主板，使用当前最新的Marlin-2.1.2.5版本进行OpenPNP配置，以下只列出部分参数，仅供参考，更多详细参数配置请查看 Marlin官方参数配置文档。

如果主板烧录了Smoothieware固件的DFU-Bootloader，只要将主板通过USB线连接上电脑后，就会将TF卡模拟成U盘，这样每次更新新编译的firmware.bin文件时，直接将固件文件复制进去，然后弹出U盘并断电重启，就能直接刷写新固件了。😀

pins_BTT_SKR_V1_4.h

定义I、J轴限位引脚

手动新增I、J轴限位引脚定义，否则Marlin在编译时会报错，提示没有定义相应的限位引脚，在实际使用时基本上用不到该引脚。

配置I、J轴步进控制引脚

将E0、E1的步进控制引脚修改为对应的I、J轴，虽然不修改Marlin会自动配置，但是我测试时发现电机只能朝一个方向，无法换向，暂不清楚是不是Marlin的BUG。💦

Configuration.h

配置信息

固件配置作者名及当前配置的类型，可视情况进行修改。

UUID设置

机器唯一ID，在某些应用程序中用于区分不同的机器。

主板型号

其他主板型号请查看Marlin固件根目录下的 Marlin/src/core/boards.h，根据自己的主板型号进行修改成对应的即可。

设置串口

BTT SKR V1.4（LPC1768）主板只有一个USB口，所以配置为-1，代表使用USB模拟串行端口。

波特率设置

过高的波特率可能会增加通讯失败的概率（受到环境干扰）。

可选波特率：
2400, 9600, 19200, 38400, 57600, 115200, 250000, 500000, 1000000

电机驱动芯片型号

对于一般的5轴贴片机来说，通常情况下需要使用5个电机驱动，所以就要指定X、Y、Z、E0、E1轴的步进电机驱动芯片型号，不同的芯片型号可以混合使用。

可选电机驱动芯片型号：
'A4988', 'A5984', 'DRV8825', 'LV8729', 'L6470', 'L6474', 'POWERSTEP01', 'TB6560', 'TB6600', 'TMC2100', 'TMC2130', 'TMC2130_STANDALONE', 'TMC2160', 'TMC2160_STANDALONE', 'TMC2208', 'TMC2208_STANDALONE', 'TMC2209', 'TMC2209_STANDALONE', 'TMC26X', 'TMC26X_STANDALONE', 'TMC2660', 'TMC2660_STANDALONE', 'TMC5130', 'TMC5130_STANDALONE', 'TMC5160', 'TMC5160_STANDALONE'

注意：
_STANDALONE 后缀表示不启用UART模式，使用STEP模式。

挤出机数量

在OpenPNP软件中，2个挤出机会被定义为2个旋转轴：C1和C2轴。

温度传感器

将板载的3个温度传感器接口都定义为998，即表示让温度恒定为25°C（不使用温度功能，可视情况修改）。

关闭挤出机PID加热控制

关闭挤出机的PID加热控制功能。

运动学

不启用任何运动学即为默认的笛卡尔运动坐标系，请根据实际机器运动学启用对应的，我的机器为CoreXY，所以启用了对应的。

硬件限位

启用X、Y、Z轴的最小行程限位功能，请根据实际机器限位位置来定义是使用最小限位还是最大限位。

限位硬件上拉

对启用的限位引脚启用上拉，以防止浮动造成的异常情况。

硬限位判定逻辑

硬限位逻辑电平判定，根据实际限位接线及触发信号进行修改。

归位限位信号检查

归位时对限位进行检查，检查是否卡住或断开。

E轴不共用参数

默认所有E轴都是共用参数的，启用后各E轴都单独使用各自的参数。

每移动1mm所需脉冲数

分别代表X、Y、Z、E轴每移动1mm需要多少脉冲，计算公式：

默认最大进给率

分别代表X，Y，Z，E0、E1轴的默认最大进给率；对于皮带传动，单位为：linear=mm/s，对于角度传动，单位为：rotational=°/s。

默认最大加速度

分别代表X，Y，Z，E0、E1轴的默认最大加速度；对于皮带传动，单位为：linear=mm/(s^2)，对于角度传动，单位为：rotational=°/(s^2)。

默认加速度

分别代表打印时默认加速度，默认回抽加速度，默认空驶加速度；对于皮带传动，单位为：linear=mm/(s^2)，对于角度传动，单位为：rotational=°/(s^2)。

线性挤出

在3D打印里该功能俗称压力提前，贴片机无需此功能，将值改为0即可。

S曲线加速度

控制加速度曲线（S曲线），可以减少机器震动和共振，电机运动更安静平稳。

Z探针

关闭Z探针功能。

喷嘴偏移

贴片机用不到该功能，都设置为0即可。

电机使能

各轴电机使能的逻辑电平，0：低电平，1：高电平。

电机方向

各轴电机旋转方向逻辑电平，true：高电平，false：低电平。

归位方向

各轴归位时的方向，-1：MIN，1：MAX。

机器尺寸

机器最大移动的距离，请按照实际情况修改。

归位坐标位置

设置归位后的坐标，对应于机器实际限位开关的位置。

最小软限位

启用对应轴的最小软限位功能，防止硬限位损坏时机器归位发生意外情况。

最大软限位

启用对应轴的最大软限位功能，防止机器移动过程中超过设定的最大值而发生意外情况。

启用详细记录

启用执行G28、G29、M48代码的详细记录，需要占用较大的flash空间，请按需考虑是否开启。

手动设置Z起始位置

对于贴片机而言，Z轴控制着2个旋转轴的高低，当Z轴归位时触发到限位后，此时2个旋转轴并不是齐平的，所以需要设置归位后，Z轴需要移动多少的距离来使2个旋转轴保持齐平，需要实测后修改。

Configuration_adv.h

配置驱动参数

此设置只对TMC驱动才有效，上述配置只列举X轴，其他轴的驱动配置以此类推；

X_CURRENT：电机驱动电流
X_CURRENT_HOME：无限位归位时的电流
X_MICROSTEPS：驱动细分数
X_RSENSE：采样电阻阻值
X_CHAIN_POS：SPI模式的CS
X_INTERPOLATE：TMC驱动插值算法（可降低打印机移动时的噪音，但是会引入少量系统位置误差）
X_HOLD_MULTIPLIER：保持电流

TMC驱动stealthChop模式

当Configuration.h -> X_DRIVER_TYPE 驱动器型号配置为支持stealthChop模式的TMC驱动时，会默认启用stealthChop模式，如果不想使用可以注释掉对应的（实测Marlin使用spreadCycle模式时，电机运行时有细微的抖动，且噪音明显，不建议启用）。
stealthChop模式：噪音更小，但是会引入系统误差。
spreadCycle模式：提供更大的扭矩和更高的精度，但是会有明显噪音。

优化斩波器参数

使用预定义的参数集优化spreadCycle斩波器参数，请按照驱动供电电压来设置正确的斩波器参数。

可选参数：
CHOPPER_DEFAULT_12V
CHOPPER_DEFAULT_19V
CHOPPER_DEFAULT_24V
CHOPPER_DEFAULT_36V
CHOPPER_09STEP_24V // 0.9 degree steppers (24V)
CHOPPER_PRUSAMK3_24V // Imported parameters from the official Průša firmware for MK3 (24V)
CHOPPER_MARLIN_119 // Old defaults from Marlin v1.1.9
